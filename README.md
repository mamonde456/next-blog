# Next.js 로 정적 블로그 만들기

### 실행

아래의 커맨드 실행

```
// git clone
git clone https://github.com/mamonde456/next-blog.git

// 설치
npm i

// 실행

npm run dev

```

### 구현

- 게시물 리스트
- 게시물 상세 페이지
- 사용자 입력란
- 입력란에 제목과 본문란 분류
- 임시글 리스트
- 임시글 재작성
